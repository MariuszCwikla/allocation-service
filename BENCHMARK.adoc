= Allocation Benchmark

Benchmark is implemented using k6 tool - `benchmark/allocation.js`.

It runs a benchmark: user requests 1 laptop and 1 monitor and measures time to allocate user's request.

== Prerequisites

1. Build `allocationservice` docker image and run service+postgres using docker compose.
2. Observe logs from `allocationservice` and wait for service to be up
3. Then run k6 benchmark

You can use just recipes:

----
just build-docker
just benchmark-setup
----

When Spring Boot application is up, simply Ctrl-C and run:

----
just benchmark
----

(or run in second terminal)
